# Posts

Drafts of public-facing communications about this project — Telegram updates, blog posts, social media drafts. Source of record for what's been said externally and when.

## Naming

`YYYY-MM-DD_<short-topic>.md`

Each file should open with a short metadata block:

- **Date** — when written
- **Channel** — where it's destined (Telegram, blog, etc.)
- **Status** — Draft / Published / Archived
- **Audience** — who it's aimed at
- **Length** — approximate character count
